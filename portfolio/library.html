Library Apps Overview

A decade ago, I played a key role in developing and designing a suite of Ruby on Rails library web applications, which positioned the library as a leader in technology adoption. I also led the usability team, conducting rigorous usability testing, generating detailed reports, and formulating strategic recommendations to enhance the overall user experience.

Library à la Carte empowered librarians to effortlessly create, manage, publish, and co-author course web pages. Through iterative usability testing and paper prototyping, librarian feedback was actively integrated, fostering familiarity and building strong buy-in for the system. Released as open-source software, Library à la Carte was successfully deployed in over a dozen libraries. Library ala Carte GitHub

Beaver Tracks, a GPS-enabled walking tour, was developed in collaboration with the University Archives to bring historical images to life. By integrating data from a Map API and historical photographs from a Photo Sharing API, the app delivers immersive, location-specific historical information.

Designed to facilitate the discovery of light and recreational reading, Book Genie offers a natural language interface where patrons can specify a subject and receive recommendations for books currently available in the library.

Although these projects are no longer active, they remain professionally significant to me.

UX/UI Research

Iterative usability testing and paper prototyping facilitated the active integration of librarian and student feedback into product development. This feedback informed the creation of content models, user flows, and wireframes. Leveraging insights from UX research, the UI design was optimized for an intuitive and accessible user experience. The UI design adhered to WCAG compliance and implemented responsive design, ensuring accessibility across all mobile devices.

Software Development

Leveraging Ruby on Rails, I developed the suite of library applications optimized for mobile responsiveness and usability. I designed efficient MySQL databases and implemented RESTful APIs to facilitate seamless data exchange with external systems. Front-end designs, built with HTML5, CSS, and JavaScript, provided consistent user experiences across diverse mobile devices. Deployed on a cloud platform, these applications ensured robust performance and scalability. The complete codebase was released as open-source, promoting community collaboration and allowing for further development and customization by other institutions.
